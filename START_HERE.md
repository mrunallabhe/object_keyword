## Quick Installation

```bash
# 1. Install all dependencies
pip install -r requirements.txt

# 2. Install CLIP (required for enhanced features)
pip install git+https://github.com/openai/CLIP.git

# 3. Run the enhanced system
python app_enhanced.py
```
